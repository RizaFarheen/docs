# Get Task Definition

The API to get all the task definitions.

## Input Payload


| Attribute  | Description                                    |
| ---------- |------------------------------------------------|
| taskType   | The name of the task for which the task definition is to be retrieved. | 
## API Endpoint
```
GET /api/metadata/taskdefs/{taskType}
```
